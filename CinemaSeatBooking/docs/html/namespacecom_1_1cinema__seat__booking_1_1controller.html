Detailed Description

REST controller for handling payment processing.

This package contains REST controllers for handling user operations in the cinema seat booking system.

REST controller providing endpoints for managing cinema seat operations.

REST controller for managing movie screening-related operations.

REST controller for managing cinema rooms and their seats.

REST controller for managing seat reservations.

Provides an endpoint to process payments associated with reservations.

Provides endpoints to create, retrieve, and cancel reservations involving users, screenings, and seats.

Provides endpoints to create, read, update, and delete rooms, along with retrieving rooms and their associated seats.

This controller provides endpoints for CRUD operations and search functionalities for movie screenings. It interacts with the ScreeningService.

This controller handles seat creation, retrieval, updating, deletion, as well as reservation and availability checking, interacting with the SeatService.

It includes UserController for managing user registration, profile management, and authentication.
