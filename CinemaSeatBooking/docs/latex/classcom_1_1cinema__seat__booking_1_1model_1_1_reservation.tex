\doxysubsection{Detailed Description}
Entity class representing a seat reservation made by a user for a screening. 

The {\ttfamily Reservation} class maps to a reservation record in the database, linking a {\ttfamily \doxylink{classcom_1_1cinema__seat__booking_1_1model_1_1_user}{User}}, a {\ttfamily \doxylink{classcom_1_1cinema__seat__booking_1_1model_1_1_screening}{Screening}}, a {\ttfamily \doxylink{classcom_1_1cinema__seat__booking_1_1model_1_1_seat}{Seat}}, and a {\ttfamily \doxylink{classcom_1_1cinema__seat__booking_1_1model_1_1_payment}{Payment}}. It tracks the reservation state and enforces cascading delete behavior.
