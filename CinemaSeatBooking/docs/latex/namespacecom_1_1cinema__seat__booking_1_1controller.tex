\doxysubsection{Detailed Description}
REST controller for handling payment processing. 

This package contains REST controllers for handling user operations in the cinema seat booking system.

REST controller providing endpoints for managing cinema seat operations.

REST controller for managing movie screening-\/related operations.

REST controller for managing cinema rooms and their seats.

REST controller for managing seat reservations.

Provides an endpoint to process payments associated with reservations.

Provides endpoints to create, retrieve, and cancel reservations involving users, screenings, and seats.

Provides endpoints to create, read, update, and delete rooms, along with retrieving rooms and their associated seats.

This controller provides endpoints for CRUD operations and search functionalities for movie screenings. It interacts with the {\ttfamily \doxylink{classcom_1_1cinema__seat__booking_1_1service_1_1_screening_service}{Screening\+Service}}.

This controller handles seat creation, retrieval, updating, deletion, as well as reservation and availability checking, interacting with the {\ttfamily \doxylink{classcom_1_1cinema__seat__booking_1_1service_1_1_seat_service}{Seat\+Service}}.

It includes {\ttfamily \doxylink{classcom_1_1cinema__seat__booking_1_1controller_1_1_user_controller}{User\+Controller}} for managing user registration, profile management, and authentication.
