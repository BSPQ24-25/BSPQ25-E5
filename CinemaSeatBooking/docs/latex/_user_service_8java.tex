\doxysubsection{Detailed Description}
Service class for managing user-\/related operations in the cinema seat booking system. 

This service provides the business logic for user management including\+:
\begin{DoxyItemize}
\item User registration with default CLIENT role
\item Authentication of users via username and password
\item Updating user profiles such as email and password
\item Deletion of user accounts with authentication check
\end{DoxyItemize}

It interacts with the {\ttfamily \doxylink{}{User\+Repository}} for persistence operations, and works closely with {\ttfamily \doxylink{}{User}} and {\ttfamily \doxylink{}{Role}} model classes.
