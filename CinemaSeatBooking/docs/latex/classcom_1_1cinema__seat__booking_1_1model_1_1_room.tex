\doxysubsection{Detailed Description}
Entity class representing a cinema room where screenings take place. 

The {\ttfamily Room} class maps to a database entity representing a cinema room. It contains a list of {\ttfamily \doxylink{classcom_1_1cinema__seat__booking_1_1model_1_1_seat}{Seat}} objects representing seats in the room and a list of {\ttfamily \doxylink{classcom_1_1cinema__seat__booking_1_1model_1_1_screening}{Screening}} objects representing scheduled screenings. It provides convenience methods to get counts of total, available, and reserved seats.
