\doxysubsection{Detailed Description}
Service interface for managing movie-\/related operations. 

The {\ttfamily Movie\+Service} interface defines the contract for CRUD operations related to the {\ttfamily \doxylink{}{Movie}} entity. Implementations of this interface handle business logic for retrieving, creating, updating, and deleting movies within the cinema seat booking application.
