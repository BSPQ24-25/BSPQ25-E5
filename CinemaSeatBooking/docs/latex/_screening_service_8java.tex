\doxysubsection{Detailed Description}
Service class responsible for managing movie screenings. 

The {\ttfamily Screening\+Service} class provides methods for retrieving, scheduling, updating, and deleting movie screenings, as well as filtering by movie, date, or location. It ensures associated seat data is initialized when needed.
